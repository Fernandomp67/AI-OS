# Memoria Maestra (Contexto Persistente)

Este archivo contiene la información fundamental que Claude debe recordar en cada sesión. Es tu "memoria a largo plazo".

## 1. Identidad Personal
- **Nombre**: Aquí debes poner tu nombre completo.
- **Profesión/Rol**: Aquí debes poner a qué te dedicas o cuál es tu rol principal.
- **Intereses Clave**: Aquí debes poner una lista de tus intereses principales (ej. IA, Desarrollo Web, Marketing, etc.).
- **Valores y Principios**: Aquí debes poner los valores que guían tu trabajo y vida.

## 2. Contexto de Negocio / Proyectos a Largo Plazo
- **Empresa/Marca**: Aquí debes poner el nombre de tu empresa o marca personal.
- **Misión**: Aquí debes poner tu misión o la de tu negocio.
- **Audiencia Objetivo**: Aquí debes poner a quién te diriges con tus proyectos.
- **Oferta Principal**: Aquí debes poner qué servicios o productos ofreces.

## 3. Reglas de Operación Preferidas
- **Tono de Voz**: Aquí debes poner cómo quieres que Claude te hable (ej. Profesional pero cercano, Directo y técnico, etc.).
- **Idioma**: Aquí debes poner tu idioma preferido para la interacción (ej. Español).
- **Formato de Salida**: Aquí debes poner si prefieres Markdown, tablas, listas, etc.

## 4. Conocimiento Crítico
- **Herramientas que usas**: Aquí debes poner una lista de software o herramientas que utilizas a diario.
- **Stack Tecnológico**: Aquí debes poner las tecnologías con las que trabajas (ej. React, Python, Supabase).
- **Restricciones**: Aquí debes poner cualquier cosa que Claude NUNCA deba hacer o decir.
