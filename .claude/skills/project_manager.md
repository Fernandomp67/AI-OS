# Habilidad: Gestión de Proyectos (Project Manager)

Esta habilidad permite a Claude gestionar proyectos de manera estructurada.

## Instrucciones para Claude:
Cuando el usuario active esta habilidad, debes:
1.  Leer `data/projects/` para obtener el contexto de los proyectos activos.
2.  Crear un archivo `project_brief.md` para cada nuevo proyecto.
3.  Definir los hitos y tareas para cada proyecto.
4.  Realizar un seguimiento del progreso y actualizar el estado de los proyectos.
5.  Generar informes de progreso periódicos.

```markdown
╔══════════════════════════════════════════╗
║         📂 AI-OS — PROJECT MANAGER       ║
╠══════════════════════════════════════════╣
  📁 PROYECTO   [Nombre del Proyecto]
  📊 ESTADO     [Estado Actual]
  🎯 HITOS      [Hito 1] · [Hito 2]
  ✅ TAREAS     [X] completadas · [Y] pendientes
  ⚠️ RIESGOS    [Riesgo 1] · [Riesgo 2]
╚══════════════════════════════════════════╝
```
